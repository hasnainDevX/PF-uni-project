# Quiz Master

A web-based quiz application built with Flask that tests your knowledge across multiple categories.

## Features

- Multiple quiz categories (Science, History, Sports, Programming, Tech History, Software & Tools, Engineering)
- Interactive UI with progress tracking
- Score calculation with visual feedback
- Answer review after completing the quiz

## Installation

1. Clone or download this project
2. Install dependencies:
```bash
python -m pip install -r requirements.txt
```

## Running the Application

1. Start the Flask server:
```bash
python app.py
```

2. Open your browser and go to:
```
http://127.0.0.1:5000
```

## How to Use

1. Select a category from the home page
2. Click "Begin Quiz" to start
3. Answer each question by selecting an option
4. Click "Next Question" to proceed
5. View your results and review answers at the end

```
